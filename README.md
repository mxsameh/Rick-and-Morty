# Rick & Morty Characters App


## Clone the project

```bash
git clone https://github.com/mxsameh/Rick-and-Morty
cd Rick-and-Morty
```

## Install dependencies
```bash
npm run install
```
## Run App
```bash
npm run dev
```
- Go to [localhost](http://localhost:5173)
